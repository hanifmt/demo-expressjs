<h1>DEMO EXPRESS.JS</h1>

<h3>Installasi</h3>
<p>1. clone git repo ke device kalian</p>
<p>2. ketik code dibawah ini pada terminal kalian</p>

```
npm install
npm run start
```

<p>3. ketik url dibawah ini pada browser kalian</p>

```
localhost:3000
```

<p>atau</p>

```
127.0.0.1:3000
```

<b>Semoga Bermanfaat</b>
